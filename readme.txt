= What it was? =

An extension for Wordpress Admin Bar plugin. Removes the header in administration and adds the missing "View site" link into the bar.

This is a plugin for a plugin for Wordpress. Extends Viper007Bond's *[Wordpress Admin Bar](http://wordpress.org/extend/plugins/wordpress-admin-bar/)*. It hides the header in administration and compensates the missing *View site* link by adding it directly into the bar. Saves about 90 px of the vertical space.

= What were the requirements? =

* Plugin *[Wordpress Admin Bar](http://wordpress.org/extend/plugins/wordpress-admin-bar/)* must be activated.
* **PHP 5 or higher** on your server is necessary to run this plugin.
